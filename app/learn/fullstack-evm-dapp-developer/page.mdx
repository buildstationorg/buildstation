# fullstack EVM dapp developer

## intro
The EVM DApp Bootcamp is a community driven initiative to train new EVM developers. EVM developers refer to developers that have sufficient knowledge to build DApps on EVM (Ethereum Virtual Machine).

## sponsor
DApp Bootcamp is sponsored by [Road to Devcon](https://blog.ethereum.org/2023/06/29/road-to-devcon7-grants).

## available languages
1. [English](/evm-dapp-bootcamp/en)
1. [Vietnamese](/evm-dapp-bootcamp/vi)

## content
Currently, EVM DApp bootcamp only covers Solidity as smart contract language and it uses an opinionated tech stack for frontend development.

## attribution
EVM DApp Bootcamp content is inspired by [Patrick Collins free course on Youtube](https://www.youtube.com/@PatrickAlphaC), along with examples from [Solidity by Example](https://solidity-by-example.org/) and [SolidityLang](https://soliditylang.org/). Content will be updated regularly.
