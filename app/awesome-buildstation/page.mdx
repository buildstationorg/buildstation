# awesome buildstation

An opinionated list of awesome projects built or contributed by the buildstation community

## wallet

- [GM GN Wallet](https://gmgn.app) - Probably the fastest onboarding wallet with Passkey.

## developer tooling

- [BlockCMD Inspect](https://blockcmd.com/inspect) - A tool to read and write to smart contracts on various blockchains.
- [GM Key](https://github.com/gmgn-app/gmkey) - A Passkey based Wallet library for the multichain world.
- [BlockCMD Kaia Foundry template](https://github.com/blockcmd/kaia-foundry-template) - A template for building smart contracts on Kaia.
- [BlockCMD Drop](https://blockcmd.com/drop) - An efficient airdrop tool on Kaia.

## education

- [DApp Bootcamp](https://github.com/buildstationorg/dapp-bootcamp) - A comprehensive bootcamp for developing DApp.
- [Cyfrin Updraft](https://updraft.cyfrin.io/) - An education platform to kickstart your smart contract development career and build real world projects, guided by industry leading expert

## trading

- [backpack-api-py](https://github.com/blockcmd/backpack-api-py) - a SDK client for Backpack Exchange API.
